Analysis of the the Canadian Aquatic Biomonitoring Network (CABIN) Parameters in Ontario

This tool allows us to evaluate the biological, chemical, habitat data (mostly in the St. Lawrence drainage area) presented in the Canadian Aquatic Biomonitoring Network Database. EPT index [Ephemeroptera (mayflies), Plecoptera (stoneflies), and Trichoptera (caddisflies)]. Map of Ecoregion can be found here Drag vertically along the axis to filter specific parameter.
